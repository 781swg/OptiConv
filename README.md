# 🖼️ ImgProc-C
### High-Performance Image Processing Library in C11

[![Language](https://img.shields.io/badge/language-C11-blue.svg)](https://en.cppreference.com/w/c/11)
[![Build](https://img.shields.io/badge/build-CMake-green.svg)](https://cmake.org/)
[![License](https://img.shields.io/badge/license-MIT-lightgrey.svg)](LICENSE)
[![Memory](https://img.shields.io/badge/memory-Valgrind--safe-brightgreen.svg)](https://valgrind.org/)

**ImgProc-C** — это мощная и модульная библиотека на языке Си для низкоуровневой обработки изображений. Она сочетает в себе математическую точность алгоритмов компьютерного зрения с высокой производительностью, характерной для системного программирования.

---

## 🌟 Ключевые особенности

*   **🧩 Модульный дизайн**: Четкое разделение ответственности. Ядро (свертки), наборы фильтров и подсистема ввода-вывода независимы друг от друга.
*   **🚀 Оптимизация**: Алгоритмы спроектированы с учетом кэш-линий процессора (Row-major access), что минимизирует задержки при обработке больших данных.
*   **🛡️ Надежность**: В проекте реализована строгая проверка всех аллокаций памяти и входных аргументов. Код "чист" от утечек памяти.
*   **📷 Широкая совместимость**: Поддержка популярных форматов (PNG, JPG, BMP) «из коробки» благодаря интеграции с `stb_image`.

---

## 🛠 Реализованные алгоритмы

| Алгоритм | Описание | Применение |
| :--- | :--- | :--- |
| **Generic Convolution** | Универсальный движок матричной свертки. | База для кастомных фильтров. |
| **Gaussian Blur** | Математически точное размытие по Гауссу. | Устранение шума, подготовка данных. |
| **Median Filter** | Нелинейная фильтрация (поиск медианы в окне). | Удаление импульсного шума ("соль и перец"). |
| **Sobel Operator** | Вычисление градиента яркости по осям X и Y. | Детекция контуров и границ объектов. |
| **Edge Clamping** | Техника обработки пикселей за границей кадра. | Предотвращение артефактов на краях. |

## 🚀 Быстрый запуск

### Зависимости
*   **Компилятор**: GCC 7.0+ или Clang
*   **Сборка**: CMake 3.10+
*   **Опционально**: Valgrind (для анализа памяти), Doxygen (для документации)

### Сборка проекта
```bash
# Клонирование и переход в папку
mkdir build && cd build
cmake ..
make
```

### Использование CLI
Утилита `imgproc_cli` позволяет обрабатывать изображения одной командой:

**Пример 1: Размытие по Гауссу (размер ядра 5, сигма 1.5)**
```bash
./imgproc_cli photo.png -gauss result.png 5 1.5
```

**Пример 2: Очистка от шума (медиана с окном 3x3)**
```bash
./imgproc_cli grainy.jpg -median cleaned.png 3
```

## 🧪 Качество и тестирование

### Модульные тесты
Для обеспечения стабильности мы используем автоматизированные тесты:
```bash
cd build
ctest --output-on-failure
```

### Анализ памяти
Проект регулярно тестируется на отсутствие утечек:
```bash
valgrind --leak-check=full ./imgproc_cli input.png -gauss out.png 3 1.0
```

### Документация
API полностью задокументировано в формате Doxygen. Для генерации локального сайта с документацией:
```bash
doxygen Doxyfile
# Откройте docs/html/index.html в браузере
```
